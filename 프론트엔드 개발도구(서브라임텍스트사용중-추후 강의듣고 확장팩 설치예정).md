# sublime text (추천 확장기능 emmet)

```
http://www.sublimetext.com/
빠르고 가볍고 확장성이 좋은 에디터
프로젝트 기능을 통해서 여러개의 폴더를 손쉽게 관리
-GOTO 기능을 통해서 원하는 파일로 빠르게 이동 화면을 쉽게 분할
-패키지 관리를 이용해서 손쉽게 확장 기능을 설치 제거
-단축키 쉬운 커스터마이징
-vi의 단축키를 기본적으로 내장해서 vi 스타일로 사용
-매크로 기능
-빌드 기능
-project->add folder to project 프로젝트 폴더추가
-project->save project as   -project 디렉토리 경로기술되어있음*제이슨 형식의 포멧
 ( '.sublime-project'라는 확장자를 가지고 있는 json 파일로 수동으로 편집도 가능)

-플러그인:
SideBarEnhancements : 사이드바에 다양한 기능을 부가하는 패키지

Goto->goto anything : 경로,리소스검색(ctrl+p), # 텍스트검색,일반검색은 파일
goto->goto simbole : @심볼검색(클래스나,메소드 네임)
goto->goto line : :1332 라인이동

ctrl+d : 다중선택(https://opentutorials.org/module/406/3629)

Sublime Package Control
메뉴 View > Show Console 을 선택해서 Console 창을 열고 아래의 명령어를 입력하고 Enter를 친다. 그리고 에디터를 재시작한다.
```
